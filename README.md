# 🔊 Amplificador de Áudio

Projeto desenvolvido na **Universidade Federal de Minas Gerais (UFMG)** para a disciplina de **Laboratório de Circuitos Eletrônicos II**.

---

## 📌 Descrição

Este projeto consiste no desenvolvimento, simulação e implementação física de um amplificador de áudio, com o objetivo de analisar seu desempenho.
